# DinoTube Video Player

Mobile-first video player experience inspired by the YouTube mobile app. The UI focuses on smooth playback, gesture-driven interactions, and a clean, responsive layout.

## Features
- Category-based video feed with thumbnails, titles, duration, and badges
- Full-page player with custom controls (play/pause, seek, Â±10s)
- Related video list filtered by category with swipe/scroll reveal
- Drag-to-minimize mini-player with persistent playback
- Auto-play next with a 2-second countdown and cancel option

## Tech Notes
The provided dataset contains YouTube URLs, which don't allow custom HTML5 controls. For the player experience, the app maps each dataset entry to a public MP4 source while preserving all titles, categories, and thumbnails.

## Getting Started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
